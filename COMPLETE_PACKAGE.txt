Bot won't start?
  1. Check .env file exists
  2. Check GROQ_API_KEY is set
  3. Check OWNER_NUMBER and ALERT_NUMBER
  4. Run: npm install
  5. Check console for error message
  6. See FIXES_SUMMARY.md for solutions

Bot won't respond?
  1. Test: !ping
  2. Check WhatsApp session (scan QR)
  3. Check internet connection
  4. Restart bot: npm start
  5. Check bot is online: !status

Commands not working?
  1. Check command syntax
  2. Use !help to see all commands
  3. Verify prefix is ! (not custom)
  4. Check GROQ_API_KEY for /ai commands

Session lost?
  1. If GITHUB_TOKEN set: data backed up
  2. If not: need to scan QR again
  3. Check session_assistant/ folder exists
  4. See README.md for session management

GitHub push failed?
  1. Check internet connection
  2. Run: git status (see changes)
  3. Run: git pull origin main (sync first)
  4. Try again: git push origin main
  5. See GITHUB_DEPLOY_GUIDE.md for help
